# Socket.io Practice


## Setup
> Runs the `npm install` and `bower install` first


## To run example

Open 2 terminal, in one run:
```bash
$ node server
```

in two run:
```bash
$ grunt serve
```

> Open multiple browsers
> Go to `localhost:9000` for all
> On one browser, click `about` and `contact` links, enter form data in contact.
